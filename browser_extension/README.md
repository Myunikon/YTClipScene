# ClipScene Browser Extension

This extension integrates your browser with the ClipScene desktop application.

## Installation Instructions

1.  **Open Extensions Page**:

    - **Chrome/Brave/Edge**: Go to `chrome://extensions`
    - **Firefox**: Go to `about:debugging` > This Firefox > Load Temporary Add-on

2.  **Enable Developer Mode**:

    - Look for a toggle switch named "Developer mode" in the top right corner and turn it on.

3.  **Load Unpacked**:
    - Click the **"Load unpacked"** button that appears.
    - Select this folder (`browser_extension`).

## Features

- **Context Menu**: Right-click on any page or link and select **"Download with ClipScene"**.
- **Video Overlay**: A "Download" button appears over `<video>` players on supported sites.
- **Deep Linking**: Automatically opens the desktop app and starts the download setup.
